# 小霸王游戏合集

## 项目拉取

```bash
git clone https://github.com/Ace627/fc-game-center.git

双击 VirtuaNES Plus 20121003.exe 即可
```

## 目录结构

```bash
|- roms                             游戏文件
|- save                             存放游戏进度记录
|- VirtuaNES Plus 20121003.exe      模拟器主程序
```

## 按键

- Z 代表 A 键
- X 代表 B 键
- Q 代表 连续 A 键
- W 代表 连续 B 键
- p 代表 解除暂停
- 空格键 代表 暂停
